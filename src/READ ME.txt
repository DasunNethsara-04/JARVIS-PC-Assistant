With default settings, Windows 10 and Windows 11 doesn't activated 'Microsoft Mark'
voice. To do it, you need to double-click on the file named 'JARVIS_Voice'.It is a
Windows Registry File.


The steps you need to follow:

1. Close the MARK 8 (MRK IIIV) Application
2. Go to 'C:\Program Files (x86)\MARK 8' (Directory of the application)
3. Open the 'src' folder
4. Double-click on the file named 'JARVIS_Voice.reg'
5. It's ready to go. Now you can run the application normally...
